What works here

  • Server-side analysis with JSON APIs.
  • Responsive frontend pages for each tool.
  • Report preview and export flow.
  • Public-safe validation for URL analysis.

How to use it

  1. Open Demo for a quick end-to-end preview.
  2. Use Website to scan a public HTTPS URL.
  3. Try Password, Crypto, and File checks.
  4. Generate a report from the Reports page.

Deployment note

The app is structured as a Flask site, so it can be hosted on platforms like Render, Railway, Fly.io, or any Python host that supports a `Procfile` and `gunicorn`.

For production, you can point a domain to the host, set `PORT` if needed, and the site will work directly from the public URL.
